# Contexto: src/

Código fuente del backend (FastAPI + Pandas).

- `components/` → piezas reutilizables del dominio (ej. modelos de datos, entidades).
- `services/` → lógica de negocio (motor de recomendación, validaciones).
- `utils/` → funciones de apoyo (lectura de Excel/CSV, formateo, etc.).
- `tests/` → pruebas automatizadas.
